# Changelog

All notable user-facing changes to Shadowtype. Newest first.

## 0.2.1 (beta)

- Language steering now reads the caret-local context instead of the whole screen, fixing wrong-language suggestions.
- Ghost-text remainder anchors correctly over accepted words on web/Electron apps.
- Stability and quality improvements to local completion.

## 0.2.0

- Selection rewrite: rewrite highlighted text in place, locally.
- Local HTTP API + MCP bridge for developers.
- Bring your own model (BYOM) support.
- Expanded model catalog (10 free swappable models).

## 0.1.0

- First beta: system-wide inline ghost-text completion, fully on-device.
- Tab / Right-Arrow to accept; per-app control.
